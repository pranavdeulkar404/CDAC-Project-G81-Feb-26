#SprintFlow: Agile Task and Bug Tracking Solution


Description:
SprintFlow is a web-based project management system designed for small software teams to manage projects, tasks, bugs, comments, and notifications in one place. The system allows users to create projects, assign tasks, report bugs, track progress, update status, and collaborate through comments. It also includes a separate notification microservice that sends alerts when tasks or bugs are assigned or updated. The project 	helps reduce dependency on spreadsheets, emails, and chat messages by providing a centralized workflow platform for team coordination. 

Software Requirements:
Frontend: React.js, HTML5, CSS3, Bootstrap / Material UI
Backend: Java, Spring Boot, Spring MVC, Spring Data JPA
Database: MySQL
Security: Spring Security with JWT Authentication
Microservice: Notification Service using Spring Boot REST API
API Integration: Mailtrap / Brevo Email API for email notifications
Tools: STS / IntelliJ IDEA, VS Code, Postman, GitHub 

Hardware Requirements:
Processor: Intel i3 or above
RAM: Minimum 4 GB, Recommended 8 GB
Storage: Minimum 10 GB free space
Operating System: Windows / Linux
Internet connection for API testing and GitHub usage
